# Klein Shell DOS 2.0 Preview

A retro-inspired command line adventure game created to recreate the experience of exploring a mysterious late-1990s computer system.

Unlike a real operating system, Klein Shell DOS is designed as a game. Players interact through classic DOS-style commands, complete missions, explore directories, read files, uncover secrets, and experience a nostalgic journey through the world of vintage computing.

## Features

* DOS-inspired command interface
* Retro 1990s computer atmosphere
* Exploration-based gameplay
* Missions and achievements
* Simulated file system
* Hidden secrets and discoveries
* Multiple visual themes
* Educational and nostalgic experience

## Project Status

This repository contains the Preview version of Klein Shell DOS 2.0.

The project is currently under development and may receive new missions, additional content, gameplay improvements, visual enhancements, and expanded story elements in future releases.

## Inspiration

Klein Shell DOS was inspired by classic DOS environments, vintage computers, bulletin board culture, and the excitement of learning computer commands during the early days of personal computing.
